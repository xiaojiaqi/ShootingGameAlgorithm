「シューティングゲームアルゴリズムマニアックス」
サンプルプログラムについて


■概要

　本CD-ROMには書籍『シューティングゲームアルゴリズムマニアックス』のサンプルプログラムが収録されています。書籍に掲載したリストと、さまざまな種類の弾や敵などを実際に動かしてみることができるデモプログラムとがあります。


■構成

○demoフォルダ
　デモプログラムです。エクスプローラなどからsht.exeを実行してください。実行にはDirectX 9.0以上と、対応ビデオカードおよびドライバが必要です。動作確認はWindowsXP/2000/98で行いました。

○listフォルダ
　書籍に掲載したリストです。オリジナルプログラムを作る際の雛形としてご利用ください。触手と多関節に関してはJava版（Java2SE1.4以上用）のサンプルも収録しました。

○projectフォルダ
　デモプログラムのプロジェクトです。Visual C++ 6.0用（sht.dsw）とVisual C++ .NET 2003用（sht.sln）のファイルを収録しました。ビルドにはDirectX 9.0 SDKをインストールして、インクルードパスやライブラリパスを設定しておく必要があります。


■デモプログラムの起動時オプション

　環境に合わせて以下のオプションを設定すると、デフォルト（800x600）よりも高い解像度でデモプログラムを実行することができます。

-w   画面の幅
-h   画面の高さ
-fw  画面の幅（フルスクリーンモード）
-fh  画面の高さ（フルスクリーンモード）

例：
sht.exe -w 1024 -h 768 -fw 1280 -fh 1024
（ウィンドウモードで1024x768、フルスクリーンモードで1280x1024に設定）
